# Fase 1 - Desafio de Lógica

## Cálculo da Ampulheta

O desafio proposto é desenhar uma ampulheta utilizando apenas JS para desenhar qualquer valor inserido pelo usuário. Para ficar mais dinâmico adicionei HTML e CSS simples.

Levei cerca de 1 hora para terminar o teste.

### O que aprendi com este desafio?
- Entendimento e aplicação em problemas de contexto
- Levantamento de hipóteses e testes das soluções
- Conhecimento em lógica de programação e estrutura de dados básica
